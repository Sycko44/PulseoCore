# Release checklist

## Versioning

- [ ] Version name updated
- [ ] Version code/build number updated
- [ ] Changelog prepared

## Infrastructure

- [ ] Production environment variables set
- [ ] Database migration plan validated
- [ ] Rollback path documented
- [ ] Healthcheck endpoint green
- [ ] Monitoring dashboards ready

## Mobile release

- [ ] App icon ready
- [ ] Screenshots ready
- [ ] Store description ready
- [ ] Privacy policy published
- [ ] Package name stable
- [ ] Signing configuration ready

## API release

- [ ] OpenAPI updated
- [ ] Breaking changes reviewed
- [ ] Client compatibility checked
- [ ] Error payloads reviewed

## Quality gates

- [ ] QA checklist passed
- [ ] Security review completed for changed sensitive areas
- [ ] Consent and privacy flows manually verified
- [ ] Crash-free smoke test completed

## Final go/no-go

- [ ] Product scope confirmed
- [ ] Known issues accepted
- [ ] Release owner assigned
- [ ] Support/rollback owner assigned
