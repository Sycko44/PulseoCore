# PulseoCore

PulseoCore is the open-source core repository for the Pulseo system.

## Project direction

This repository follows these rules:

- open source first;
- self-hostable on a VPS;
- no mandatory proprietary dependency for core operation;
- public documentation for architecture, contracts, and delivery;
- the current product source of truth is **Pulseo 1.2**.

## What is already in this repository

The `docs/autopilot-pack/` directory contains the current pilot pack:

- canon freeze;
- master progress register;
- mobile release spec;
- design system canon;
- junior execution pack;
- lot 33 semantic sandbox library;
- 90-day execution roadmap.

## Immediate goals

1. build the Android MVP connected to the VPS;
2. publish the mobile release spec and real API contracts;
3. make the design system executable;
4. ship a junior-friendly execution pack;
5. implement the first MVP of the semantic sandbox library.

## Open-source baseline

This repository is intended to remain:

- forkable;
- auditable;
- rebuildable;
- documented.
