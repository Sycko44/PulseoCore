# apps/mobile

Android/mobile client for the first Pulseo MVP.

## Expected scope

- onboarding
- auth
- home
- check-in
- cravings
- SOS
- rituals
- Phoenix
- profile/settings

## Rules

- consume the root `openapi.yaml` contract
- keep secrets out of the client
- support staging and production API base URLs
- implement loading, error, empty and offline states
